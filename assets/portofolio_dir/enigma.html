Description

Enigma is an application that I developed to encrypt and decrypt messages, with a focus on privacy and security of communications. Using a peer-to-peer (P2P) structure, the app allows users to share encrypted messages and grant access to other specific users. The operation of Enigma is based on the configuration of a local server through ngrok, which establishes an API for communication between users. This API is shared with authorized users, allowing them to access and communicate securely through encrypted messages. To ensure the security of communication, strong cryptographic algorithms were used, such as SHA-256 and AES encryption. These algorithms provide an additional level of protection for messages in transit and ensure that only authorized users can decrypt and read messages. In short, Enigma is an application that combines strong encryption techniques, a P2P structure, and a local server to offer a secure and private platform for encrypted communication between users.
