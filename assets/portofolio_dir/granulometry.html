Description

Granulometry is an innovative application that I have developed and that is part of my programming portfolio. This application is designed to facilitate the sieving process in laboratories, eliminating the need for manual calculations or paper notes.
With Granulometry, users can enter stuck values ??on sieves used in the soil sieving process for both fine and coarse soils. The application automatically takes care of performing the necessary calculations and provides an accurate and fast result.
One of the main advantages of Granulometry is its ability to speed up and simplify the sieving process in laboratories. Users no longer have to rely on error-prone manual calculations or pen and paper. Instead, the app provides them with a reliable and efficient solution.
Granulometry's interface has been designed with usability and user experience in mind. It is intuitive and easy to navigate, allowing users to enter stuck values ??quickly and easily. In addition, the application displays the results in a clear and legible way, which facilitates the interpretation of the data obtained.
