Description

Notes is an application that he programmed and that is based on the concept of a social network. It allows users to post and share messages with each other, thus creating an interactive and collaborative platform.

The application has a strong user system, where each user can create an account and access their personalized profile. This allows secure identification and authentication of users.

For the implementation of the backend, Flask, a lightweight and flexible web framework in Python, has been used. Flask provides a solid foundation for the development of microservices, which allow for a modular and scalable architecture. These microservices handle different functionalities of the application, such as user management, message posting, and user interaction.

In addition, JWT (JSON Web Tokens) has been integrated to guarantee security in the communication between the client and the server. JWT is a widely used security standard that enables authentication and the secure transmission of information in the form of tokens.

Notes is a showcase of my ability to develop applications with a solid backend foundation using modern technologies and scalable approaches. By including this project in my portfolio, I highlight my experience in creating robust and secure web applications.

In the development of Notes, a MySQL database has been used to efficiently store and manage information related to users, messages, and other application entities. MySQL is a widely used and reliable database management system that provides strong performance and great scalability. The integration of MySQL into the project ensures efficient and secure data management, providing a seamless experience for Notes users.

I am proud to have programmed Notes, and I hope that its inclusion in my portfolio demonstrates my ability to develop interactive and functional applications, using modern technologies to provide a satisfying and secure user experience.
